# Task Manager Frontend

This is the **frontend application** for the MoveoAI Backend Engineer Challenge.

It is built to consume and interact with the official backend API:

🌟 https://github.com/antonellafernandez/MoveoAI2026BackendEngineerChallenge

## Technologies Used

- React
- TypeScript
- React Router
- TanStack React Query
- Axios
- Tailwind CSS

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Configure environment variables

Create a .env file:
```bash
VITE_API_URL=http://localhost:8080
```

### 3. Run the app

```bash
npm run dev
```
